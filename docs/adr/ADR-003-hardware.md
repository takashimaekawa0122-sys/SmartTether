# ADR-003: ターゲットハードウェア選定

- **日付**: 2026-03-22
- **ステータス**: 決定済み

## 決定事項
**Xiaomi Smart Band 9** を採用する

## 理由
- BLEプロトコルがGadgetbridgeによって完全解析・公開済み
- 振動コマンドの実装実績が豊富
- Amazon.co.jpで入手可能（¥8,150）
- バイブレーション機能が確認済み

## 却下した選択肢
- **LivelyLife R99**: バイブレーション機能なし
- **Xiaomi Smart Band 10**: BLEプロトコル解析が未完成（2026年3月時点）
- **Xiaomi Smart Band 8**: Amazon.co.jpで在庫なし

## 注意事項
- Xiaomiのファームウェア更新でBLEプロトコルが変わる可能性あり
- Auth Keyは家族のAndroidスマホを使って取得する（一度だけ）
- Band 9のバッテリー残量を定期取得し、電池切れ時の誤検知を防ぐ
